Sources for "The Enemy and How I Helped to Fight it" by Shlomi Fish

  • English DocBook 5 - source in DocBook 5 (XML format). Can be edited directly.
  • English XHTML - to be read - generated from the docbook. DO NOT EDIT DIRECTLY!.
  • Hebrew Fiction-Text source - a well-formed plaintext format (see the XML-Grammar-Fiction homepage) that is the prototypal source of the Hebrew version. Can be edited directly.
  • Hebrew XHTML - as generated indirectly from the Hebrew Fiction-Text. DO NOT EDIT DIRECTLY!, but can be used to view the page comfortably.
  • The Makefile - used to build the sources, and upload them. Requires some dependencies.

The preferable way to edit these sources is to clone the Bitbucket repository of the story, and send me a pull request.
